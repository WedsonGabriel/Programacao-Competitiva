# 🏆 Programação Competitiva

Este repositório é dedicado a armazenar as resoluções de problemas, algoritmos e desafios realizados durante a minha preparação para maratonas de programação competitiva. Este projeto acompanha o meu desenvolvimento prático e lógico no curso de Análise e Desenvolvimento de Sistemas (TADS) no IFRN.

Todo o material, treinamento e progresso documentados aqui contam com a orientação do **Prof. Jorgiano Vidal**.

## 🎯 Foco e Competições

O objetivo principal deste repositório é centralizar os estudos e resoluções voltados para as maiores competições de programação do Brasil:
*   **SBC** (Maratona de Programação da Sociedade Brasileira de Computação)
*   **OBI** (Olimpíada Brasileira de Informática)

## 💻 Linguagem Utilizada

*   **C++**: Linguagem exclusiva deste repositório, escolhida por sua alta performance, eficiência na execução de algoritmos complexos sob pressão de tempo e pelo uso extensivo da Standard Template Library (STL).
